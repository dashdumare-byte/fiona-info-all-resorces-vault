# Devin Desktop (Exclusion Reaffirmed)

> **Status: EXCLUDED -- Original "Devin AI" exclusion extended to Devin Desktop.**

---

## What Changed

**June 2, 2026**: Cognition rebranded Windsurf IDE as Devin Desktop.

This is NOT the same "Devin AI" we originally excluded. That was the cloud autonomous agent. Devin Desktop is an IDE with multi-agent management. But we still exclude it.

---

## Why Still Excluded

| Issue | Detail |
|-------|--------|
| **Cloud-only models** | Every model runs on Cognition's servers. No local weights. |
| **Full overlap** | Agent Command Center = Pi + Goose + Orchestrator. DeepWiki = GitNexus + Obsidian. |
| **Extra cost** | $20-200/mo subscription. FIONA budget is for API calls, not IDE subscriptions. |
| **Wrong paradigm** | Devin is for human engineers delegating to agents. FIONA IS the agent. |
